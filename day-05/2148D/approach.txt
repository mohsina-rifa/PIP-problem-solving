URL: https://codeforces.com/problemset/problem/2148/D
difficulty level : 1000







SENARIO:

field-i   :   array[i] flower

odd-flower  :   machine on  : turn off
                machine off : turn on     : cut flowers
otherwise   :   machine on  : cut flowers
                machine off : walk away




APPROACH:
Greedy

=> partition array    :   even array
                          odd array

=> field > 0    :   update answer   :   sum of all evens

=> calculate odd fields to take
      take the largest odd value    :   maximize the total sum.
