difficulty level : 1700







RULE:

white flower  : group of size k





FIND:

no of ways to eat flower





APPROACH:
Dynamic-Programming

=> define values  : modulo
                    max possible sequence length
                    arrays    :   dp-table
                                  prefix sum

=> DP table:
      dp[i]   :   no. of valid sequences  :   length = i.
      
      prefix sum array  :   for fast range queries.

      red flower      :   dp[i] += dp[i-1]
      K white flowers :   dp[i] += dp[i-K]  : if i >= K
